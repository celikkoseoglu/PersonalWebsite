My Projects

QNote

Blazing fast note taking.

Makes note taking fast, simple and easy on Windows Phone devices. Unlike other note taking apps, QNote does not require the user to wait for a few seconds to get ready for note taking. It starts up in milliseconds, the user takes some notes and just locks the phone to close the application. Perfect for taking quick notes on the go!

Technologies Used: Microsoft Visual Studio C# .NET WPF
Download for: Windows Phone Windows

PhotoSec

Secure your photos!

Select some photos from the phone’s gallery and password protect them. PhotoSec uses multi-threading to display and render photos much faster when compared to other similar applications in the store. Also, it was the 36th most downloaded application from the Windows Phone Store. PhotoSec was written in C# XAML.

Technologies Used: Microsoft Visual Studio C# .NET WPF
Download for: Windows Phone
